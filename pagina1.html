INTRODUCUCCION AL DISEÑO DE SITIOS WEB CON GIT.

Objetivo general: Brindar información básica sobre el manejo de herramientas que se enfocan al desarrollo web como lo son GIT, GITHUB, HTML5, CSS3, mediante videos, lecturas, y las actividades que se vendrán realizando durante el curso.

Objetivo específico 1: Aprender conceptos y aplicaciones del diseño de sitios web.

Objetivo específico 2: introducción al diseño de sitios web, enfocándonos en el funcionamiento del sistema de control de versiones GIT y el uso de GITHUB para poder manejarlos de forma adecuada.

Objetivo específico 3: Manejo y creación de un sitio web con total seguridad y profesionalismo.
